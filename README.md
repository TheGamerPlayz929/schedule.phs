# PHS Student Portal

A public student portal for Poolesville High School schedule information, announcements, privacy notes, and the embedded GradeViewer entrypoint.

## Live Site

- https://poolesville.web.app/

## Public Pages

- `index.html` - schedule and day status
- `announcements.html` - public announcements
- `gradeviewer.html` - GradeViewer entrypoint
- `privacy.html` - privacy and safety notes

## Release Notes

This repository contains only public-facing static assets and Firebase Hosting configuration. Development-only tooling, local server code, internal security notes, legacy experiments, and private operational docs should not be committed here.
